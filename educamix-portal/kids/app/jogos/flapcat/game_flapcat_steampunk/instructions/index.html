A) The game - Top
The game Game FlapCat Steampunk was developed using the program Construct 2, the Scirra. The game can be viewed on almost all internet browsers. Included is touch, and can be opened in phones and tablets.

The game also is auto-resizable! The game was developed in resolution 1280x720px (HD). You can use any size in height and width, but it gets better in the original resolution. Another suggestion is 860x480px resolution, which also in this file.

Tip: Always use the latest version of the program more stable Construct.

C) Embed a game into your website - Top
To embed a game in your site is very simple, just copy the folder to your server. The folder with the game ready is this:



1 - Copy the folder to your server. Feel free to rename the folder.
2 - In this application example of the game, it resizes to the size of longs player. It is advisable to keep the default size of 1280x720px but you can use the full screen game, because it adjusts to any size.
3 - I added the size of 860x480px, because this size is easier to put on most sites.
D) Game with iframe - Top
Try to use your site in the game with iframe, to experience the best player. In this mode the game keeps the original dimensions.

In the file that you have purchased contains an iframe ready for you to use! The page ready in this directory:



If you want to change the iframe page, follow the steps below:

1 - Copy the folder with the game on the resolution chosen folder to your server. Decorate the directory name, because you'll need to put in the iframe.
2 - On the page that you want the game to appear, copy the following code in html:

<iframe src="gamefolder/index.html" width="860px" height="480px"> </ iframe>

3 - Where is "gamefolder" you put the entire directory to where you put the game on your server, example: "games/adventure/flapcat".
4 - If you want to change the size of the game will appear on your website, simply modify the "width" and "height". You can use whatever size you want, because the game will adjust alone.
E) Editing images - Top
To edit the images of the game, no need to leave the program. Just replace them. Follow the steps below.

1 - The pictures are in this directory: game_flapcat_steampunk/game/images or game_flapcat_steampunk/game_iframe/game/images



2 - Make sure the image you want to replace has the same dimensions of the original.



3 - Always save in PNG, because if you save as jpeg, it will not work, because it lacks transparency.
4 - Replace the original image you created.
5 - Okay, now try the game!

Important: You should always make a backup before editing, because you may want to recover the original file.

G) Share buttons - Top
Now everyone will be able to share your game. FlapCat has 2 buttons for sharing, one for Facebook and one for Twitter.

To edit the text that will appear in Facebook and Twitter, just open the XML file in Dreamweaver, Notepad or the editor you prefer.

Important: Create a Facebook app, is free. The app ID that will be used in XML to sign the post to Facebook. Check out how to create your app:

1 - When you are logged in, check the arrow in the corner and click "Create App".



2 - You will be in the developer's Facebook page. Go on the top menu and click on Apps, then "Create New App".



3 - Enter the name of the app you want to appear, and also a single name.



4 - Okay, now just divide the number on your ID, so you can share your game on your behalf. Just below you will learn how to use this ID.



If you DO NOT WANT to use the buttons on your game, just save an image 1px transparent images replace of the original buttons.

All XML data can be directly entered into the Construct program. So, do not need the XML file. Learn more on the topic Game without xml how.

The XML file was created for those who have the Construct program can pass information to share for the game. You can find the XML file inside the directory: game_flapcat_steampunk/game/xml/your_game.xml



1 - Open the file your_game.xml may be in DreamWeaver, Notepad or the editor of your choice.

2 - Enter your App ID from Facebook. It is a sequence with several núemros. You learn to create an ID in the Share buttons.



2 - Item 2 and 3 are the key phrases that will appear in the post. TXT RECORD you place a text that comes before the record, as in the example below:



3 - Among <name> tags you put the game name or any text you prefer.



4 - Among the <link> tags you put the link of your game, or link from your site.



5 - Here you put a subtitle to your post.



6 - Write a very beautiful image of your game, preferably square. It can be any size. I used 80x80px. You put the image on your site or on any server online. Then copy the image link below and place. This image is very important, so it is important to be very beautiful.



7 - Insert here the description of your game.



8 - From here are the Twitter data. Enter below the text that will come before the value of the record.



9 - After the record, you can put text and a link to your game.



Okay, now just save and test online.
