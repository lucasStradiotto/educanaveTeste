A) The game - Top
The game Indiara and the Gold Skull was developed using the program Construct 2, the Scirra. The sounds, music and layout were all developed specifically for the game, and you can change them as you wish.
B) 2 resolutions game - Top
The game adapts according to the size you want, but even then I did with two resolutions, 640x480px and 840x480px.

The lower resolution of 640x480px is squarer, and will not appear very level. But a resolution is widely used and recommended for gaming.

Advise at the time of entering the game in your site, use the resolution 840x480px. It appears more level, more graphics, and the field of view increases the pleasure of playing.
C) Embed a game into your website - Top
To embed a game in your website is very simple, just use an iframe on the page that you want the game to appear. Follow the steps below:

1 - The games are in this directory: indiara-and-the-skull-gold/game. The games with iframe ready with a bg done this in this folder: indiara-and-the-skull-gold/game_iframe
2 - Copy the folder with the game on the resolution chosen folder to your server. Decorate the directory name, because you'll need to put in the iframe.
3 - On the page that you want the game to appear, copy the following code in html:

<iframe src="gamefolder/index.html" width="840px" height="480px"> </ iframe>

4 - Where is "gamefolder" you put the entire directory to where you put the game on your server, example: "games/adventure/indiara".
5 - If you want to change the size of the game will appear on your website, simply modify the "width" and "height". You can use whatever size you want, because the game will adjust alone.

For the game appears alone on the page itself, just use their own "index.html" which is in the game folder.

E) Editing images - Top
To edit the images of the game, no need to leave the program. Just replace them. Follow the steps below.

1 - The pictures are in this directory: indiara-and-the-skull-gold/game/640x480px/images and indiara-and-the-skull-gold/game/840x480px/images
2 - Make sure the image you want to replace has the same dimensions of the original.
3 - Always save in PNG, because if you save as jpeg, it will not work, because it lacks transparency.
4 - Replace the original image you created.
5 - Okay, now try the game!

Important: You should always make a backup before editing, because you may want to recover the original file.
